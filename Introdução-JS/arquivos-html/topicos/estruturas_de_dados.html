Estruturas de Dados

Arrays

Arrays são coleções ordenadas de valores que podem ser de qualquer tipo, como números, strings, objetos, ou até outros arrays. Eles são definidos usando colchetes [], com os elementos separados por vírgulas. Por exemplo, const frutas = ["maçã", "banana", "laranja"];. Arrays possuem um índice numérico que começa em 0, o que significa que o primeiro elemento é acessado com frutas[0].

Objetos

Objetos são coleções de pares chave-valor, onde as chaves são strings (ou símbolos) que mapeiam para valores de qualquer tipo. Eles são definidos usando chaves {}. Por exemplo, const pessoa = { nome: "Ana", idade: 25 };. Você pode acessar as propriedades usando a notação de ponto pessoa.nome ou colchetes pessoa["idade"].

Podemos pensar em um objeto como um array porem voce tem que nomear os indices agora chamados de propriedades.

Acessando propriedades usando ponto

Usado quando você conhece o nome da propriedade e ela é um identificador válido de JavaScript (sem caracteres especiais ou espaços).

Nesse caso a propriedade é tratado como literal

Literal: No contexto de programação, literal é um valor escrito diretamente no código, representando exatamente aquilo que você vê. É um valor fixo, que não depende de variáveis ou cálculos.

Não pode ser usado se o nome da propriedade:

  • Contém espaços ou caracteres especiais (exemplo: marca do carro).
  • Começa com um número (exemplo: 1modelo).
  • É uma palavra reservada em JavaScript (exemplo: class).

Acessando propriedades usando colchetes

Usado quando você precisa acessar propriedades dinamicamente, ou seja, o nome da propriedade não é um literal fixo, mas armazenado em uma variável ou definido como string. Além disso permite qualquer string como nome da propriedade, mesmo com caracteres especiais, espaços ou números.

Métodos Básicos de Arrays

  • push()

    • Adiciona um ou mais elementos ao final do array.

    Retorno: o número de elementos no array após a adição.

  • pop()

    • Remove e retorna o último elemento do array.

  • map()

    • O método map() em JavaScript é usado para criar um novo array com os resultados da aplicação de uma função em cada elemento do array original. Ele não modifica o array original.

    Retorno: Um novo array com os valores transformados pela função fornecida.

  • reduce()

    • Reduz um arrys a um único resultado/valor acumulando ao longo do array.

    Retorno: um unico valor resultande da função usada.

    No caso do método reduce() a função que ele recebe terá o primeiro parâmetros como o valor valor Anterior e o segundo como valor Atual. Você vai usar esses parâmetros na função que irá considerar cada array um valor.

    Após a função você deve definir qual será o primeiro valor do primeiro parâmetros, pois ainda não há valor anterior.

    Como os dois primeiros parâmetros da função em reduce são referidos aos elementos anteriores e atual, os parâmetros seguintes seguem a ordem normal sendo o terceiro o indice e o quarto o array.

    na última iteração, o valor do acumulador é atualizado com base no último elemento do array, e esse valor final é o que o reduce()

  • filter()

    • Filtra os elementos de um arrays com base em uma condição e retorna um novo array com os elementos que passaram no teste.

  • shift()

    • Remove e retorna o primeiro elemento do array.

  • unshift()

    • Adiciona um ou mais elementos no início do array.

    Tanto o método shift quanto o unshift, ao serem usados em um array modificam a quantidade de indices e os indices de cada elemento.

  • splice()

    • Para remover, adicionar ou substituir elementos no array original.

    OBS: ele modifica o array original adicionando ou removendo os elementos

    Seus parametros serve para especificar onde comeca a exclusão de elementos e quantos serão excluidos a partir do primeiro parametro . Os demais adicionados serão considerados elementos a serem adiconados nesse espaço que foi apagado.

    array.splice(índiceInicial, quantidadeParaRemover, item1, item2, ..., itemN)

    Se você colocar 0 no segundo parametro ele nao irá apagar nada, porém se ele for omitido todos os itens a partir do indiceInicial serão removidos.

    O método splice retorna apenas elementos removidos

  • slice()

    • Para criar cópias de partes de um array sem alterar o array original.

    Você terá que especificar onde irá comecar e terminar a parte que você quer cópiar do array original. O primeiro valor será onde comeca e o segundo onde para.

    Se você não especificar nenhum parametro ele irá copiar o array inteiro.

  • some()

    • Verifica se pelo menos um elemento do array passa no teste, retornando "true" ou "false".

  • every()

    • Verifica se todos os elemento do array passam no teste, retornando "true" ou "false".

  • find()

    • Retorna o primeiro elemento que passa em um teste especifico.

  • include()

    • Verifica se um array tem um valor especifico, retornando "true" ou "false".

  • sort()

    • Ordena os elementos do arrys de forma crecente ou decrecente

    Esse método tem 2 parametros na sua função callback e os compara se der negativo ele mantem a ordem se der positivo ele troca os valores se der zero mantem a ordem

    voce pode usar a funcao 'localeCompare' para compara strings essa funcao compara duas strings de acordo com as regras do idioma, retornando um valor +, - ou nulo.

  • forEach

    • Serve para iterar e executar funções em cada elemento do array

    O método em si não retorna nada (Undefined), porém ele serve para você iterar sobre os elementos do array e armazenar ou exibir eles.

    Caso queria modificar algo no array original é necessario que faça explicitamente isso como no exemplo acima.

Explicação dos 3 parâmetros da função:

Elemento:

Representa o valor atual do array que está sendo processado. É o elemento em que a função está "trabalhando" no momento.

Indice:

Representa o índice (posição) do elemento atual no array. Útil se você precisa saber a posição do elemento para algum cálculo ou lógica.

Array:

Representa o array completo que está sendo iterado. Útil se você precisa acessar o array inteiro dentro da função.

OBS: vale lembar que os nomes dos parâmetros são nomeados, ou seja, você pode escolher os nomes. oque importa no final é a orndem que eles foram colocados.

A depender do método ele pode precisar de um ou mais parametros assim, ou nem sequer necessitar da função callback (usada dentro do método).

Como excluir elementos de um array

O método mais comum e eficaz para remover um elemento em um índice específico é o splice(). Pórem existe outro muito usado também chamado 'Delete'.

Delete em array

O delete remove o elemento de um índice específico, mas não reordena o array. O índice ainda existirá, mas será preenchido com undefined.

Obs:O uso de delete não é recomendado para arrays, porque deixa "buracos" no array.

Heranças em JavaScript

Em JavaScript, cada objeto tem um prótotipo (prototype), que é outro objeto do qual ele 'herda' propriedades e métodos. Quando você tenta acessar um propriedade ou método em um objeto, o JavaScript primeiro verificar se essa propriedade existe no proprio objeto. Se não encontrar , ele então procura no prototipo do objeto, e assim por diante até chegar no topo que é (Object.prototype).

OBS:Se você mudar algum protótipo os objetos que herdam dele também verão as mudanças.

E se você mudar uma propriedade ou método herdado no objeto, essa propriedade deixara de ser herdada e vai ser do proprio objeto e a propriedade do protótipo permanece inalterada

Se quiser "voltar" a usar a propriedade herdada, remova a propriedade própria com delete. E se você deletar a propridade no protótipo, ela é deletada do obejto que tinha ele como protótipo.

This em JavaScript

O this no JavaScript é uma palavra-chave especial que referencia o contexto de execução atual, ou seja, o objeto ao qual a função está associada ou que a "invocou". O valor de this depende de como e onde a função é chamada.

No contexto global, fora de qualquer função ou objeto, o valor de this depende do ambiente:

  • Navedador:

    • Em navegadores, this referencia o objeto global window.

  • Node.js:

    • Em módulos Node.js, this referencia o objeto module.exports.

  • Dentro de uma função comum:

    • this em funções regulares referencia o objeto global (como window no navegador).
