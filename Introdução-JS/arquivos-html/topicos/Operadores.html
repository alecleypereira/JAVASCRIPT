Operadores do JAVASCRIPT

O que são Operadores

Operadores são símbolos especiais em programação que são usados para realizar operações em operandos (valores ou variáveis). Eles permitem a manipulação e a comparação de dados de forma eficiente e clara. Cada tipo de operador tem uma função específica e é essencial para a construção de expressões e instruções em um programa.

Operadores Aritméticos

Os operadores aritméticos são usados para realizar operações matemáticas comuns.

  • Adição (+): a + b
  • Subtração (-): a - b
  • Multiplicação (*): a * b
  • Divisão (/): a / b
  • Módulo (%): a % b (resto da divisão)
  • Incremento (++): a++ (aumenta o valor em 1)
  • Decremento (--): a-- (diminui o valor em 1)

Operadores de Atribuição

Os operadores de atribuição são usados para atribuir valores às variáveis.

  • Atribuição Simples (=): a = b
  • Atribuição de Adição (+=): a += b (equivale a a = a + b)
  • Atribuição de Subtração (-=): a -= b (equivale a a = a - b)
  • Atribuição de Multiplicação (*=): a *= b (equivale a a = a * b)
  • Atribuição de Divisão (/=): a /= b (equivale a a = a / b)
  • Atribuição de Módulo (%=): a %= b (equivale a a = a % b)

Operadores Relacionais

Os operadores relacionais são usados para comparar valores e retornam um booleano (true ou false).

  • Igualdade (==): a == b
  • Desigualdade (!=): a != b
  • Estritamente Igual (===): a === b
  • Estritamente Desigual (!==): a !== b
  • Maior que (>): a > b
  • Maior ou Igual (>=): a >= b
  • Menor que (<): a < b
  • Menor ou Igual (<=): a <= b

Operadores Lógicos

Os operadores lógicos são usados para combinar múltiplas condições booleanas.

  • E (&&): a && b (true se ambos os operandos forem true)
  • OU (||): a || b (true se pelo menos um dos operandos for true)
  • NÃO (!): !a (inverte o valor booleano)

Operador Ternário

O operador ternário é uma forma abreviada de escrever uma declaração condicional if-else.

  • Sintaxe: condição ? valor1 : valor2
  • Se condição for true, o operador retorna valor1, caso contrário, retorna valor2.

Importância dos Operadores

Operadores são fundamentais na programação porque permitem a execução de cálculos, a tomada de decisões, a atribuição de valores e a combinação de expressões complexas de maneira clara e concisa. Sem operadores, a escrita e a leitura de código se tornariam muito mais complicadas e menos intuitivas.

Operadores Avançados em JavaScript

Operadores de Comparação Estrita e Não Estrita

Em JavaScript, os operadores de comparação são usados para comparar valores e retornam um valor booleano (true ou false). Existem dois tipos principais de operadores de comparação: estritos e não estritos.

Comparação Não Estrita

Os operadores de comparação não estritos (== e !=) comparam dois valores após convertê-los para um tipo comum (coerção). Isso significa que, se os valores a serem comparados forem de tipos diferentes, o JavaScript tentará convertê-los para um tipo comum antes de fazer a comparação. Essa conversão pode levar a resultados inesperados.

Comparação Estrita

Os operadores de comparação estritos (=== e !==) comparam tanto o valor quanto o tipo dos operandos sem realizar qualquer conversão de tipo. Isso garante que a comparação seja feita de maneira mais precisa e previsível.

Usar operadores de comparação estrita (=== e !==) é geralmente considerado uma boa prática em JavaScript, pois evita comportamentos inesperados causados pela coerção de tipos.

Operadores Unários

  • typeof: Retorna o tipo de dado de uma variável.
  • delete: Remove uma propriedade de um objeto.
  • void: Avalia uma expressão e retorna undefined.
  • +: Converte o operando para um número.
  • -: Converte o operando para um número e aplica a negação.
  • ++ e --: Incrementa ou decrementa o valor da variável.
