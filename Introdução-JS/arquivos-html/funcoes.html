Funções

O que são Funções

Funções são blocos de código que realizam uma tarefa específica e podem ser reutilizadas em diferentes partes do código.

Você pode declarar uma função usando a palavra-chave function, seguida pelo nome da função, parênteses e um bloco de código.

Parâmetros, Argumentos e Retorno

  • Parâmetros: São variáveis listadas como parte da definição da função. Eles atuam como placeholders para os valores que você passará para a função.
  • Argumentos: São os valores reais passados para a função quando ela é chamada.
  • Retorno: As funções podem retornar valores usando a palavra-chave return. Isso permite que a função envie um resultado de volta ao local onde foi chamada.

Conceitos Avançados

  • Funções Anônimas: Funções sem nome, frequentemente usadas como argumentos para outras funções ou para criar funções temporárias.
  • Funções de Ordem Superior: Funções que podem receber outras funções como argumentos ou retorná-las.
  • Funções Arrow (Setas): Uma forma mais concisa de definir funções.
  • Funções Imediatamente Invocadas (IIFE): Funções que são executadas assim que são definidas.
